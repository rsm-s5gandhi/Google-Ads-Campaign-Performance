# Google Ads Campaign Performance Dashboard

An interactive Tableau dashboard analyzing Google Ads campaign performance across multiple dimensions including KPIs, monthly trends, device-level insights, and campaign state breakdown.

---

## 🚀 Project Overview

This dashboard was built to answer a key business question:

### Where is performance actually coming from?

It provides:

- Executive-level KPI summary (Clicks, CTR, Conversions, Cost, Avg CPC, Cost/Conv)
- Monthly performance trends
- Device-level breakdown (Android vs iOS)
- Campaign State analysis (Enabled, Paused, Removed)
- Top campaign performance comparison

The focus of this project was not just visualization — but accurate metric modeling and structured dashboard design.

---

## 📐 Key Design & Analytical Decisions

- ✔ Used weighted CTR (Total Clicks / Total Impressions) instead of averaging row-level CTR
- ✔ Corrected cost values from micro-units scaling
- ✔ Applied consistent visual hierarchy for executive readability
- ✔ Designed using a modern dark theme for clarity and contrast
